About Angular Dashboard Fluance

Angular Dashboard Fluance is an implenent of the dashboard design pattern for angularjs. It define a common base to build a strong and smart dashboard interface.
Only defining Models-Controllers at a low level, it is CSS framework free, and build to feet any project.
